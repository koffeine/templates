# Koffeine's project templates

## Usage

```sh
npx degit koffeine/templates/<template> <dir>

cd <dir>

git init
git add
```

## Templates

- basic: `npx degit koffeine/templates/basic`
- node: `npx degit koffeine/templates/node`
